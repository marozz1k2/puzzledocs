# PuzzleDocs

Краткая документация по API для пользователей PuzzleAI. Здесь только практические инструкции: какой метод выбрать, какие поля заполнить и что вернётся в ответ.

## Быстрый старт

1. Выберите нужную статью по модели в навигации.
2. Скопируйте спецификацию запроса (GET/POST).
3. Передайте базовые параметры: `prompt`, `bot`, `user`.
4. Отправьте запрос в no-code интеграции и проверьте ответ.
