# PuzzleDocs

Краткая документация по API для пользователей PuzzleAI. Здесь только практические инструкции: какой метод выбрать, какие поля заполнить и что вернётся в ответ.

## Быстрый старт трекера

1. В команде PuzzleBot добавьте действие **«Отправить запрос»**.
2. Укажите ссылку `https://api.pxsto.re/main/puzzlebot-tracker`.
3. Выберите метод `POST` и вид запроса `Сформированный`.
4. Передайте обязательную базу: `bot`, `token`, `user`, `model`.
5. Добавьте содержимое задачи: чаще всего `prompt`, а для медиа-моделей ещё `images` или `params`.

## Минимальный пример с комментариями

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии.
  "model": "gpt_5", // обязательно: ключ модели, которую нужно вызвать.
  "prompt": "{{prompt}}", // обычно обязательно: текст задачи; исключения описаны в статье модели.
  "send_answer": true // необязательно: отправить результат пользователю; по умолчанию `true`.
}
```

## Базовые параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота. Используйте `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота. |
| `user` | string | Да | ID пользователя или сессии. Используйте `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Ключ модели из нужной статьи, например `gpt_5`, `gpt_image`, `veo` или `openai_gpt_5_4_pro` из каталога OpenRouter. |
| `prompt` | string | Зависит от модели | Текст задачи. Для большинства моделей обязателен. |
| `images` | string | Зависит от модели | Ссылки, file_id или переменные с изображениями через запятую, без массива `[]`. |
| `params` | object | Нет | Вложенные настройки конкретной модели. |
| `role` | string | Нет | Дополнительная роль или инструкция для модели. |
| `send_answer` | boolean | Нет | `true` отправляет результат в чат, `false` сохраняет его в `{{tracker_answer}}`. |
| `chat` | string/number | Нет | ID чата или канала для отправки результата. |
| `topic` | number | Нет | ID топика форума. |

## Puzzle AI Tracker

Puzzle AI Tracker позволяет владельцам Puzzle AI плагина собирать собственные AI-сценарии в PuzzleBot: текстовые ответы, генерацию изображений, видео, аудио, документов и медиа-обработку.

Главное правило: сначала передайте обязательную базу, затем добавляйте только параметры выбранной модели. Необязательные поля не нужно создавать в запросе, пока они реально не используются.
