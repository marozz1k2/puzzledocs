# PuzzleDocs

Документация для сервиса PuzzleAI в формате GitBook.

## Что находится в этом репозитории

Этот репозиторий содержит:

- документацию по трекер-запросам;
- API-референсы по моделям;
- описание параметров и переменных в формате `{{VAR}}`;
- структуру по группам моделей:
  - Текст
  - Голос
  - Фото
  - Видео
  - Музыка
  - Другое

## Базовый endpoint

Для большинства запросов используется единая точка:

- `https://api.pxsto.re/main/puzzlebot-tracker`

## Принципы оформления статей

Каждая статья оформляется в стиле API reference и должна содержать:

1. Краткое описание модели и сценариев использования.
2. Endpoint и метод запроса.
3. Список параметров (обязательные/необязательные).
4. Пример запроса в формате no-code API (ключ/значение).
5. Пример ответа (если доступен).
6. Примечания по лимитам, ошибкам и best practices.
