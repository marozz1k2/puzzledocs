# Основная логика

#### Что мы разберем в этом разделе

Мы не будем изучать все функции конструктора PuzzleBot. Мы сосредоточимся только на тех инструментах, которые необходимы для запуска стабильного ИИ-сервиса:

* **Ограничение доступа:** использование категорий для защиты от спама.
* **Логика баланса:** создание переменных и условий для списания запросов.
* **Автоматизация продаж:** подключение платежных систем.
* **Интерфейс**: создание команд для комфортной работы пользователя с нейросетью.

***
