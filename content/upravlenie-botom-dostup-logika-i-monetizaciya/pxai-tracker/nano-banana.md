# Nano Banana

Функционал предназначен для продвинутых пользователей, которые хотят получить полный контроль над редактированием изображений через Google Nano Banana (модель `nano_banana`).

**Для чего это нужно?**

Хотя Nano Banana поддерживается в Puzzle AI по умолчанию, наш трекер поможет:

* Создавать кастомные мини-приложения для продвинутого редактирования фото.
* Гибко управлять процессом, используя несколько изображений одновременно.
* Встроить Nano Banana в различные сценарии (например, "виртуальная примерочная" или "фотостудия").

#### Настройка запроса

В любой команде вашего бота (например, "`banana_tracker`") добавьте действие «Отправить запрос» и настройте его:

* Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
* Тип запроса: `POST`
* Вид запроса: `Сформированный`

### Практические примеры

Вы можете использовать трекер для создания полноценных продуктов, ботов и мини-приложений на базе Nano Banana.

#### Кейс 1. Бот для "виртуальной примерочной"

**Задача:** Пользователь загружает свой портрет (Изображение 1) и одежду, которую хочет примерить (Изображение 2). Бот должен "надеть" эту одежду на пользователя, сохранив его лицо.

**Решение:**

1. Создайте команду, например, `/tryon`.
2. В ней сделайте две «Формы ввода» подряд:
   * Форма 1 (Фото):
     * Текст: `Загрузите свое фото (желательно в полный рост или портрет).`
     * Тип ввода: `Отправка сообщения`
     * Маска ввода: `Изображение`
     * Переменная: `{{user_photo}}`
   * Форма 2 (Фото):
     * Текст: `Загрузите фото с новым образом` &#x20;
     * Тип ввода: `Отправка сообщения`
     * Маска ввода: `Изображение`
     * Переменная: `{{outfit}}`

<figure><img src="../../.gitbook/assets/image (221).png" alt=""><figcaption></figcaption></figure>

3. Сразу после нее — действие «Отправить запрос» (Tracker) со смешанными параметрами:

| Ключ     | Значение                                                      | Зачем?                                                   |
| -------- | ------------------------------------------------------------- | -------------------------------------------------------- |
| `images` | `{{user_photo}},{{outfit}}`                                   | Берем фото пользователя и одежды.                        |
| `prompt` | `Надень на человека эту одежду, сохрани его лицо и прическу.` | Комбинируем ваш "мастер-промпт" с запросом пользователя. |
| `model`  | `nano_banana`                                                 | Указываем модель Nano Banana.                            |
| (и т.д.) |                                                               |                                                          |

**Результат:** Бот создает фото в новом образе. Nano Banana идеально сохраняет лицо пользователя, чего не могут другие модели.

#### Кейс 2. Бот для удаления фона

**Задача:** Пользователь загружает фото (например, селфи в комнате) и хочет просто убрать фон.

**Решение:**

1. Создайте команду, например, `/background`.
2. Сделайте новую «Форму ввода» (Фото):
   * Текст: `Загрузите фото, где нужно убрать фон.`
   * Тип ввода: `Отправка сообщения`
   * Маска ввода: `Изображение`
   * Переменная: `{{user_photo}}`

<figure><img src="../../.gitbook/assets/image (220).png" alt=""><figcaption></figcaption></figure>

3. В действии «Отправить запрос» (Tracker) укажите:

| Ключ     | Значение                                                  | Зачем?                                                       |
| -------- | --------------------------------------------------------- | ------------------------------------------------------------ |
| `images` | `{{user_photo}}`                                          | Фото от пользователя.                                        |
| `prompt` | `Удали фон на этой фотографии. Больше ничего не меняй`    | Ваш "мастер-промпт", который гарантирует сохранение объекта. |
| `model`  | `nano_banana`                                             | Указываем модель Nano Banana.                                |
| (и т.д.) |                                                           |                                                              |

**Результат:** Пользователь получает свое фото, но уже без фона.

<figure><img src="../../.gitbook/assets/gandr-collage - 2025-10-20T153652.439.jpg" alt=""><figcaption></figcaption></figure>
