# GPT Image

Функционал предназначен для продвинутых пользователей, которые хотят получить полный контроль над созданием и преобразованием изображений через GPT Image. Основной ключ модели: `gpt_image`. Для явного выбора версии также доступны `gpt_image_2` и `gpt_image_1_5`.

**Для чего это нужно?**

Хотя GPT Image поддерживается в Puzzle AI по умолчанию, наш трекер поможет:

* Создавать кастомные мини-приложения для генерации и стилизации фото.
* Гибко управлять процессом, используя исходные изображения как основу для перерисовки.
* Встроить GPT Image в различные сценарии (например, "AI-аватар" или "генератор рекламных креативов").

#### Настройка запроса

В любой команде вашего бота (например, "gpt\_image\_tracker") добавьте действие «Отправить запрос» и настройте его:

* Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
* Тип запроса: `POST`
* Вид запроса: `Сформированный`

### Практические примеры

Вы можете использовать трекер для создания полноценных продуктов, ботов и мини-приложений на базе GPT Image.&#x20;

#### Кейс 1. Бот "Студия аватаров" (стилизация)

**Задача:** Пользователь загружает свое селфи. Бот должен полностью перерисовать это селфи в разных художественных стилях (например, "Киберпанк", "Аниме 90-х", "3D-персонаж Pixar").

**Решение:**

1. Создайте команду, например, `/avatar`.
2. В ней сделайте две «Формы ввода» подряд:
   * Форма 1 (Фото):
     * Текст: `Загрузите свое селфи (желательно крупным планом).`
     * Тип ввода: `Отправка сообщения`
     * Маска ввода: `Изображение`
     * Переменная: `{{user_selfie}}`
   * Форма 2 (Текст):
     * Текст: `В каком стиле сделать аватар? (например: "киберпанк-герой", "аниме-персонаж", "герой GTA").`
     * Переменная: `{{style_prompt}}`

<figure><img src="../../.gitbook/assets/image (62).png" alt=""><figcaption></figcaption></figure>

3. Сразу после нее — действие «Отправить запрос» (Tracker) со смешанными параметрами:

| **Ключ**       | **Значение**                                                     | **Зачем?**                                   |
| -------------- | ---------------------------------------------------------------- | -------------------------------------------- |
| `images`       | `{{user_selfie}}`                                                | Берем селфи пользователя как основу.         |
| `prompt`       | `Полностью перерисуй человека на фото в стиле {{style_prompt}}.` | Комбинируем "мастер-промпт" с выбором стиля. |
| `model`        | `gpt_image`                                                      | Указываем модель GPT Image.                  |
| `aspect_ratio` | `2:3`                                                            | (Статически) Делаем портретный аватар.       |

**Результат:** Бот создает совершенно новое изображение, которое похоже на пользователя, но выполнено в заданном художественном стиле.

<figure><img src="../../.gitbook/assets/gandr-collage - 2025-10-20T180814.740.jpg" alt=""><figcaption></figcaption></figure>

#### **Кейс 2. Бот "Визуализатор скетчей" (для художников)**

**Задача:** Художник или дизайнер загружает свой быстрый карандашный набросок (скетч). Бот должен "додумать" его и превратить в фотореалистичный концепт-арт.

**Решение:**

1. Создайте команду, например, `/concept`.
2. Сделайте одну «Форму ввода» (Фото):
   * Текст: `Загрузите свой скетч или набросок.`
   * Тип ввода: `Отправка сообщения`
   * Маска ввода: `Изображение`
   * Переменная: `{{user_sketch}}`

<figure><img src="../../.gitbook/assets/image (63).png" alt=""><figcaption></figcaption></figure>

3. В действии «Отправить запрос» (Tracker) укажите статический промпт:

| Ключ     | Значение                                                                                                              | Зачем                                                                  |
| -------- | --------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| `images` | `{{user_sketch}}`                                                                                                     | Фото скетча от пользователя.                                           |
| `prompt` | `Преврати этот карандашный набросок в детализированный, фотореалистичный концепт-арт. Добавь цвета, тени и текстуры.` | Ваш "мастер-промпт", который задает задачу. Пользователь его не видит. |
| `model`  | `gpt_image`                                                                                                           | Указываем модель GPT Image.                                            |

**Результат:** Пользователь получает высокодетализированный арт, основанный на его собственной композиции. GPT Image использует скетч как "карту" для создания нового изображения.

<figure><img src="../../.gitbook/assets/gandr-collage - 2025-10-20T181536.597.jpg" alt=""><figcaption></figcaption></figure>
