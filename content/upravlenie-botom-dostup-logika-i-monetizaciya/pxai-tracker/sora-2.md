# Sora 2

Функционал предназначен для продвинутых пользователей, которые хотят получить полный контроль над созданием видео через Sora 2.

**Для чего это нужно?**

Хотя Sora 2 поддерживается в Puzzle AI по умолчанию, наш трекер поможет:

* Создавать кастомные мини-приложения для генерации видео.
* Гибко управлять параметрами (качество, длительность, ориентация).
* Встроить Sora 2 в различные сценарии

#### Настройка запроса

В любой команде вашего бота (например, "`sora_tracker`") добавьте действие «Отправить запрос» и настройте его:

* Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
* Тип запроса: `POST`
* Вид запроса: `Сформированный`

### Практические примеры

Вы можете использовать трекер для создания полноценных продуктов. Вы даете пользователю одну кнопку, а всю сложную работу (промпт и настройки) прячете «под капот».

#### Кейс 1. Бот для дизайнера интерьеров

**Задача:** Пользователь загружает фото пустой комнаты, а бот сам обставляет ее мебелью и анимирует. Пользователь не вводит текст.

**Решение:**

1. Создайте команду, например, `/design_room`.
2. В ней сделайте одну «Форму ввода»:
   * Текст: `Загрузите фотографию пустой комнаты, и я создам анимированный дизайн-проект.`
   * Тип ввода: `Отправка сообщения`
   * Маска ввода: `Изображение`
   * Переменная: `{{room_photo}}`

<figure><img src="../../.gitbook/assets/image (223).png" alt=""><figcaption></figcaption></figure>

3. Сразу после нее — действие «Отправить запрос» (Tracker) со смешанными параметрами:

| Ключ     | Значение                                                                                                                           | Зачем?                                                                                         |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| `images` | `{{room_photo}}`                                                                                                                   | Берем фото комнаты от пользователя.                                                            |
| `prompt` | `animate this room, add photorealistic modern furniture (sofa, table, plant), bright daytime lighting, cinematic smooth pan right` | Это ваш «мастер-промпт». Вы сами его придумали и жестко вшили. Пользователь его даже не видит. |
| `model`  | `sora-2-pro`                                                                                                                       | Вы сами решаете, что нужна pro-версия.                                                         |
| и т.д.   |                                                                                                                                    |                                                                                                |

**Результат:** Ваш пользователь получает готовый, анимированный дизайн-проект, просто загрузив одно фото. Он не  парился с промптами и настройками. Всю сложную работу вы, как разработчик, сделали за него.

#### Кейс 2. E-commerce / «Оживление» товара

Этот кейс идеально подходит, если вы хотите создать услугу для малого бизнеса (например, для магазинов).

**Задача:** Владелец магазина загружает 1 фотографию своего товара (например, кроссовки, сумку, часы). Бот должен автоматически создать профессиональный 3D-видеоролик, вращающий этот товар, для использования в рекламе или Reels. Пользователь не вводит текст.

**Решение:**

1. Создайте команду, например, `/product_video`.
2. В ней сделайте одну «Форму ввода» для загрузки фото:
   * Текст: `Загрузите 1 фотографию вашего товара (желательно на белом фоне), и я создам из нее рекламный ролик.`
   * Тип ввода: `Отправка сообщения`
   * Маска ввода: `Изображение`
   * Переменная: `{{product_photo}}`

<figure><img src="../../.gitbook/assets/image (224).png" alt=""><figcaption></figcaption></figure>

3. Сразу после нее — действие «Отправить запрос» (Tracker) со смешанными параметрами:

| Ключ          | Значение                                                                                                                   | Зачем?                                                                   |
| ------------- | -------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| `images`      | `{{product_photo}}`                                                                                                        | Берем фото товара от пользователя.                                       |
| `prompt`      | `cinematic 360-degree rotation of this product, professional studio lighting, on a clean minimal pedestal, photorealistic` | Ваш «мастер-промпт» для создания 3D-вращения. Пользователь его не видит. |
| `model`       | `sora-2-pro`                                                                                                               | Выбираем HD-модель для лучшего качества.                                 |
| `orientation` | `portrait`                                                                                                                 | Сразу готовим ролик для формата Reels/TikTok.                            |
| (и др.)       |                                                                                                                            |                                                                          |

**Результат:** Ваш пользователь получил готовый рекламный ролик. Вы, как разработчик, продаете не просто "доступ к Sora", а готовое решение для бизнеса.

***

Puzzle AI Tracker отлично подходит для создания сложных и кастомных решений.

Используя комбинацию:

* Динамических данных от пользователя (переменных `{{sora_prompt}}`, `{{sora_image}}`)
* Статических параметров (жестко прописанных промптов, `orientation: portrait` или `size: large`)

...вы можете перейти от простого AI-бота к созданию полноценных продуктов (как в кейсах с дизайнером интерьеров или e-commerce). Вся сложность остается под капотом, а ваш пользователь получает готовое решение в одну кнопку.

Экспериментируйте с этими параметрами, чтобы создавать по-настоящему мощные и полезные инструменты.
