# Персональная очистка истории диалога с GPT

Эта инструкция поможет вам создать в боте команду, которая принудительно очищает историю переписки конкретного пользователя с нейросетью GPT.

Для чего это может быть полезно?

* **Кнопка "Новый диалог"**. Вы можете дать пользователю возможность самому начинать разговор с "чистого листа", чтобы предыдущий контекст не влиял на новые ответы GPT.
* **Сброс при ошибках.** Если нейросеть "зациклилась" или начала отвечать некорректно, сброс истории может решить проблему.
* **Административные действия.** Вы, как администратор, можете вручную сбросить диалог для любого пользователя.

**Настройка команды для очистки истории**

1. **В конструкторе PuzzleBot создайте новую команду** с любым понятным названием, например, `Сбросить диалог GPT`.

<figure><img src="../../.gitbook/assets/image (162).png" alt=""><figcaption></figcaption></figure>

2. **В созданной команде добавьте действие "Отправить запрос"** и настройте его следующим образом:

* Ссылка: `https://api.pxsto.re/main/history-clear-personal`
* Тип запроса: `POST`
* Вид запроса: `Сформированный`

3. **Добавьте в запрос три обязательных параметра.**

* Ключ: user\
  Значение: `{{USER_ID_TEXT}}`
* Ключ: bot\
  Значение: `{{BOT_USERNAME_TEXT}}`
* Ключ: token\
  Значение: \[Ваш API-токен]  (Вставьте сюда ваш API-токен для входящих запросов из раздела `Настройки > Интеграции`)

<figure><img src="https://telegra.ph/file/2bcd7c3ffa148635a9130.png" alt=""><figcaption><p>Персональная очистка диалога пользователя с GPT</p></figcaption></figure>

После выполнения этой команды история диалога указанного пользователя с GPT будет полностью очищена.<br>
