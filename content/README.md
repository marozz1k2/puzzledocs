# Введение

PxAI помогает добавить ИИ в вашего бота на конструкторе PuzzleBot — без серверов, сложных API и разработки с нуля.

Ваш бот в Telegram или MAX сможет отвечать через AI-модели, генерировать изображения, видео, музыку и выполнять другие AI-задачи, а вы сможете управлять доступом, лимитами и списанием запросов.

{% hint style="info" %}
PxAI работает только совместно с конструктором PuzzleBot
{% endhint %}

### Какие задачи решает PxAI

PxAI нужен, чтобы встроить AI-функции в уже существующего или нового бота на PuzzleBot и управлять ими без разработки собственной технической инфраструктуры.

**С помощью PxAI вы можете:**

* подключать текстовые AI-модели;
* добавлять генерацию изображений, видео, музыки и другие AI-инструменты;
* настраивать роль и поведение AI-модели;
* подключать бота к группам и бизнес-аккаунтам/

### Как это работает

Общий принцип работы выглядит так:

1. вы создаёте и настраиваете бота в PuzzleBot;
2. подключаете бота к PxAI;
3. выбираете нужные AI-модели и инструменты;
4. задаёте правила их работы: роль, лимиты, стоимость запросов, реакции и ограничения;
5. добавляете ИИ-функции в нужные команды, сценарии или мини-приложения.

После этого пользователи продолжают взаимодействовать с вашим ботом в Telegram или MAX. Они отправляют сообщения, нажимают кнопки, запускают команды или используют мини-приложения.

Когда в сценарии нужен ИИ, запрос передаётся в PxAI. Сервис обращается к выбранной AI-модели, получает результат, возвращает его в вашего бота и применяет заданные правила списания.
