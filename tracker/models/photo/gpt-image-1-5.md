# gpt_image_1_5

Отдельный ключ GPT Image 1.5 для генерации и редактирования изображений с поддержкой референсов.

## Как отправлять запрос

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Метод: `POST`
- Вид запроса в PuzzleBot: `Сформированный`
- Формат тела: `application/json` или те же ключи в no-code параметрах.

Обязательная база для любого запроса: `bot`, `token`, `user`, `model`. Для редактирования или работы по референсам добавьте `images`.

## Пример запроса с комментариями

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота для доступа к трекеру.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии.
  "model": "gpt_image_1_5", // обязательно: точный ключ модели.
  "prompt": "{{prompt}}", // обязательно: описание изображения или правки.
  "images": "https://example.com/reference.jpg", // необязательно: ссылки, file_id или переменные через запятую, без массива `[]`.
  "params": { // необязательно: вложенные настройки модели.
    "quality": "high", // необязательно: качество результата.
    "aspect_ratio": "1:1", // необязательно: соотношение сторон.
    "size": "1024x1024", // необязательно: размер результата.
    "background": "auto", // необязательно: фон результата.
    "output_format": "png" // необязательно: формат файла результата.
  }
}
```

## Параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота. В PuzzleBot обычно используется `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота для авторизации запроса. |
| `user` | string | Да | ID пользователя или сессии. В PuzzleBot обычно используется `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Точный ключ вызываемой модели: `gpt_image_1_5`. |
| `prompt` | string | Да | Описание изображения или правки. |
| `images` | string | Нет | Входные изображения/референсы через запятую. |
| `params.quality` | string | Нет | `medium` или `high`; по умолчанию в текущей конфигурации используется `high`. |
| `params.aspect_ratio` | string | Нет | `1:1`, `2:3` или `3:2`, если параметр поддерживается выбранным провайдером. |
| `params.size` | string | Нет | Размер результата, например `1024x1024`, `1024x1536`, `1536x1024` или `auto`. |
| `params.background` | string | Нет | Фон результата: `auto`, `opaque` или `transparent`, если выбранный провайдер поддерживает прозрачность. |
| `params.output_format` | string | Нет | Формат результата: `png`, `jpeg` или `webp`. |
| `send_answer` | boolean | Нет | `true` отправляет результат пользователю. `false` сохраняет результат в `{{tracker_answer}}`. По умолчанию `true`. |

## Ответ

```json
{
  "answer": "Ссылка на готовое изображение."
}
```
