# suno

`suno` — музыкальная модель для генерации песен, инструментальных треков и обработки аудио. Один и тот же ключ модели используется для нескольких методов: простой генерации, кастомной генерации, кавера, продления и разделения трека.

## Как отправлять запрос

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Метод: `POST`
- Вид запроса в PuzzleBot: `Сформированный`
- Формат тела: `application/json` или те же ключи в no-code параметрах.

Обязательная база для любого запроса: `bot`, `token`, `user`, `model`. Для генерации добавьте `prompt`, нужный метод укажите в `mode`, а версию Suno — в `submodel`.

## Универсальный пример с комментариями

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота для доступа к трекеру.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии, к которой относится запрос.
  "model": "suno", // обязательно: точный ключ модели.
  "submodel": "v5.5", // необязательно: версия Suno внутри модели; используйте доступную в вашем аккаунте версию.
  "prompt": "{{suno_prompt}}", // обычно обязательно: описание трека, текст песни или инструкция для метода.
  "instrumental": "{{suno_music_only}}", // необязательно: `true`, если нужен трек без вокала; `false`, если можно использовать вокал.
  "styles": "{{suno_styles}}", // необязательно: стили или жанры через запятую, например `pop, lo-fi`.
  "mode": "{{suno_mode}}", // необязательно: `simple`, `custom`, `extend` или `separate`; по умолчанию используйте `simple`.
  "gpt_lyric": "{{suno_gpt_lyric}}", // необязательно: `true`, если текст песни нужно сначала сгенерировать через GPT.
  "additional": "{{suno_additional}}", // необязательно: расширенные настройки Suno, например версия, голос и прочие параметры интерфейса.
  "gpt_tags_edit": true, // необязательно: разрешить GPT улучшить теги стиля перед отправкой в Suno.
  "extend": "{{suno_extend}}", // условно обязательно: точка продолжения для метода `extend`.
  "send_answer": true // необязательно: отправить результат пользователю; по умолчанию `true`.
}
```

## Параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота. В PuzzleBot обычно используется `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота для авторизации запроса. |
| `user` | string | Да | ID пользователя или сессии. В PuzzleBot обычно используется `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Точный ключ вызываемой модели: `suno`. |
| `submodel` | string | Нет | Версия Suno внутри модели. Подробнее: [suno-submodels.md](suno-submodels.md). |
| `prompt` | string | Зависит от метода | Описание трека, текст песни или инструкция. Для простой и кастомной генерации обязателен. |
| `instrumental` | boolean/string | Нет | `true` создаёт инструментал без вокала. `false` оставляет возможность вокала. |
| `styles` | string | Нет | Стили, жанры и теги через запятую. Пример: `trance, ambient, cinematic`. |
| `mode` | string | Нет | Метод работы: `simple`, `custom`, `extend`, `separate`. Если поле не задано, используйте поведение простого режима. |
| `gpt_lyric` | boolean/string | Нет | `true` просит GPT написать текст песни по идее из `prompt`. |
| `additional` | string/object | Нет | Расширенные настройки из интерфейса Suno: версия, голос, качество, креативность и другие параметры. |
| `gpt_tags_edit` | boolean | Нет | `true` разрешает GPT доработать музыкальные теги. |
| `extend` | string | Условно | Нужен только для `mode=extend`: время или позиция, с которой нужно продолжить трек. |
| `send_answer` | boolean | Нет | `true` отправляет результат пользователю. `false` не отправляет сообщение и сохраняет результат в `{{tracker_answer}}`. По умолчанию `true`. |

## Переменные сценария PuzzleBot

Эти значения используются в готовом no-code сценарии, но не все из них являются отдельными ключами запроса трекера.

| Переменная | Где используется | Описание |
| --- | --- | --- |
| `{{suno_prompt}}` | `prompt` | Финальное описание трека или текст песни. |
| `{{suno_styles}}` | `styles` | Выбранные стили музыки. |
| `{{suno_mode}}` | `mode` | Выбранный метод работы. |
| `{{suno_gpt_lyric}}` | `gpt_lyric` | Флаг генерации текста песни через GPT. |
| `{{suno_music_only}}` | `instrumental` | Флаг инструментального трека. |
| `{{suno_additional}}` | `additional` | Интегрированная переменная с расширенными настройками. |
| `{{suno_cover}}` | сценарий загрузки аудио | Загруженный файл для кавера, продления или разделения. В финальном списке параметров трекера отдельный ключ для него не указан. |
| `{{suno_extend}}` | `extend` | Время или позиция для продления трека. |

## Ответ

```json
{
  "answer": "Ссылка на готовый аудиофайл."
}
```

Если `send_answer = false`, результат не отправляется пользователю в чат, а сохраняется в переменную `{{tracker_answer}}`.
