# flow_music

`flow_music` — музыкальная модель Flow Music для генерации треков и работы с аудио. Используйте её как отдельный `model`, не как submodel Suno.

## Как отправлять запрос

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Метод: `POST`
- Вид запроса в PuzzleBot: `Сформированный`
- Формат тела: `application/json` или те же ключи в no-code параметрах.

Обязательная база для любого запроса: `bot`, `token`, `user`, `model`. Для генерации добавьте `prompt`; для обработки загруженного аудио дополнительно используйте переменную с файлом в вашем сценарии PuzzleBot.

## Пример запроса с комментариями

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота для доступа к трекеру.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии, к которой относится запрос.
  "model": "flow_music", // обязательно: точный ключ модели Flow Music.
  "submodel": "lyria_3", // необязательно: подмодель Flow Music, если нужно выбрать конкретный движок.
  "prompt": "{{flow_music_prompt}}", // обязательно: описание трека, текст песни или задача для аудио.
  "mode": "generate", // необязательно: метод работы, например `generate`, `remix`, `extend` или `separate`.
  "styles": "{{flow_music_styles}}", // необязательно: стили и жанры через запятую.
  "duration": 60, // необязательно: желаемая длительность результата в секундах, если поддерживается выбранным методом.
  "instrumental": false, // необязательно: `true`, если нужен трек без вокала.
  "lyrics": "{{flow_music_lyrics}}", // необязательно: готовый текст песни, если он хранится отдельно от `prompt`.
  "send_answer": true // необязательно: отправить результат пользователю; по умолчанию `true`.
}
```

## Параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота. В PuzzleBot обычно используется `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота для авторизации запроса. |
| `user` | string | Да | ID пользователя или сессии. В PuzzleBot обычно используется `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Точный ключ модели: `flow_music`. |
| `submodel` | string | Нет | Подмодель Flow Music. Для Lyria используйте `lyria_3`, если она доступна в вашем аккаунте. |
| `prompt` | string | Да | Описание музыки, текст песни или инструкция к обработке аудио. |
| `mode` | string | Нет | Метод работы: `generate`, `remix`, `extend`, `separate`. Если поле не задано, используйте обычную генерацию. |
| `styles` | string | Нет | Стили, жанры и теги через запятую. |
| `duration` | number | Нет | Желаемая длительность результата в секундах, если выбранный метод поддерживает длительность. |
| `instrumental` | boolean/string | Нет | `true` создаёт инструментал без вокала. |
| `lyrics` | string | Нет | Готовый текст песни, если его нужно передать отдельно от общего `prompt`. |
| `send_answer` | boolean | Нет | `true` отправляет результат пользователю. `false` сохраняет результат в `{{tracker_answer}}`. |

## Методы

| Метод | Значение `mode` | Что делает |
| --- | --- | --- |
| Генерация | `generate` | Создаёт новый трек по текстовому описанию. |
| Remix | `remix` | Перерабатывает загруженное аудио по новой инструкции. |
| Продление | `extend` | Продолжает загруженный трек. |
| Разделение | `separate` | Разделяет загруженный трек на вокал и инструментал, если метод доступен в аккаунте. |

## Минимальный запрос

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии.
  "model": "flow_music", // обязательно: ключ модели Flow Music.
  "prompt": "{{flow_music_prompt}}" // обязательно: описание трека или текст песни.
}
```

## Ответ

```json
{
  "answer": "Ссылка на готовый аудиофайл."
}
```

Если `send_answer = false`, результат не отправляется пользователю в чат, а сохраняется в переменную `{{tracker_answer}}`.
