# suno: продление трека

Метод продления продолжает загруженный трек с выбранной точки. Пользователь сначала загружает аудио, затем выбирает время продолжения.

В готовом no-code сценарии аудио сохраняется в `{{suno_cover}}`, а точка продолжения — в `{{suno_extend}}`.

## Как отправлять запрос

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Метод: `POST`
- Вид запроса в PuzzleBot: `Сформированный`
- Формат тела: `application/json` или те же ключи в no-code параметрах.

## Пример запроса с комментариями

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота для доступа к трекеру.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии, к которой относится запрос.
  "model": "suno", // обязательно: точный ключ музыкальной модели.
  "submodel": "v5.5", // необязательно: версия Suno внутри модели.
  "prompt": "{{suno_prompt}}", // рекомендуется: описание того, как продолжить трек.
  "mode": "extend", // обязательно: режим продления трека.
  "extend": "{{suno_extend}}", // обязательно: время или позиция, с которой нужно продолжить трек.
  "styles": "{{suno_styles}}", // необязательно: стили и жанры для продолжения.
  "instrumental": "{{suno_music_only}}", // необязательно: `true`, если продолжение должно быть без вокала.
  "additional": "{{suno_additional}}", // необязательно: расширенные настройки Suno из интегрированной переменной.
  "gpt_tags_edit": true, // необязательно: разрешить GPT доработать теги стиля.
  "send_answer": true // необязательно: отправить результат пользователю; по умолчанию `true`.
}
```

## Параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота. В PuzzleBot обычно используется `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота для авторизации запроса. |
| `user` | string | Да | ID пользователя или сессии. В PuzzleBot обычно используется `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Точный ключ модели: `suno`. |
| `submodel` | string | Нет | Версия Suno внутри модели: `v3`, `v3.5`, `v4`, `v4.5`, `v5`, `v5.5`. |
| `prompt` | string | Нет | Инструкция, настроение или текст для продолжения трека. |
| `mode` | string | Да | Для продления передайте `extend`. |
| `extend` | string | Да | Время или позиция продолжения, например значение из переменной `{{suno_extend}}`. |
| `styles` | string | Нет | Музыкальные стили через запятую. |
| `instrumental` | boolean/string | Нет | `true` указывает, что продолжение должно быть инструментальным. |
| `additional` | string/object | Нет | Расширенные настройки Suno: версия, голос и другие параметры интерфейса. |
| `gpt_tags_edit` | boolean | Нет | `true` разрешает GPT улучшить музыкальные теги. |
| `send_answer` | boolean | Нет | `true` отправляет результат пользователю. `false` сохраняет результат в `{{tracker_answer}}`. |

## Переменные сценария

| Переменная | Обязательная | Описание |
| --- | --- | --- |
| `{{suno_cover}}` | Да | Загруженный аудиофайл для продления. Это переменная сценария PuzzleBot, а не отдельный ключ из финального списка параметров трекера. |
| `{{suno_extend}}` | Да | Точка продолжения трека. Это значение передаётся в параметр `extend`. |

## Ответ

```json
{
  "answer": "Ссылка на готовый аудиофайл."
}
```
