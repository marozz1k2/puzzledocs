# Музыкальные модели

Музыкальные модели используются для генерации песен, инструментальных треков и обработки уже загруженного аудио.

## Как отправлять запросы

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Метод: `POST`
- Вид запроса в PuzzleBot: `Сформированный`
- Формат тела: `application/json` или те же ключи в no-code параметрах.

Минимальная база всегда одна: `bot`, `token`, `user`, `model`. Для музыки почти всегда добавляется `prompt`. Конкретное действие выбирается через `mode`, а версия внутри модели — через `submodel`, если модель это поддерживает.

## Базовый пример

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии.
  "model": "suno", // обязательно: ключ музыкальной модели.
  "submodel": "v5.5", // необязательно: версия внутри модели, если она поддерживает submodel.
  "prompt": "{{suno_prompt}}", // обычно обязательно: описание трека, текст песни или задача для обработки.
  "mode": "simple", // необязательно: метод работы модели; если не нужен другой режим, используйте `simple`.
  "send_answer": true // необязательно: отправить результат пользователю; по умолчанию `true`.
}
```

## Базовые параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота: `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота. |
| `user` | string | Да | ID пользователя или сессии: `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Ключ музыкальной модели: `suno` или `flow_music`. |
| `submodel` | string | Нет | Версия или подмодель внутри выбранной модели, например версия Suno или Lyria для Flow Music. |
| `prompt` | string | Зависит от метода | Описание музыки, текст песни или инструкция к обработке аудио. |
| `mode` | string | Нет | Метод работы модели. Для Suno: `simple`, `custom`, `extend`, `separate`. Для Flow Music: `generate`, `remix`, `extend`, `separate`. |
| `send_answer` | boolean | Нет | `true` отправляет результат в чат, `false` сохраняет его в `{{tracker_answer}}`. |

## Методы Flow Music

- Генерация нового трека: `mode=generate`
- Remix загруженного аудио: `mode=remix`
- Продление трека: `mode=extend`
- Разделение на вокал и инструментал: `mode=separate`
