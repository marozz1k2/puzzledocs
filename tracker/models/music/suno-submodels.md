# suno: submodels

`submodel` выбирает версию Suno внутри модели `suno`. Это не отдельный `model`: в запросе всегда остаётся `"model": "suno"`, а версия передаётся отдельным параметром.

## Как отправлять запрос

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Метод: `POST`
- Вид запроса в PuzzleBot: `Сформированный`
- Формат тела: `application/json` или те же ключи в no-code параметрах.

## Пример запроса с комментариями

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота для доступа к трекеру.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии, к которой относится запрос.
  "model": "suno", // обязательно: основной ключ музыкальной модели.
  "submodel": "v5.5", // обязательно для выбора версии: submodel Suno, доступный в вашем аккаунте.
  "prompt": "{{suno_prompt}}", // обязательно: описание трека, текст песни или задача для генерации.
  "mode": "custom", // необязательно: метод работы Suno; можно использовать `simple`, `custom`, `extend` или `separate`.
  "styles": "{{suno_styles}}", // необязательно: стили и жанры через запятую.
  "send_answer": true // необязательно: отправить результат пользователю; по умолчанию `true`.
}
```

## Доступные submodels

| Submodel | Когда использовать |
| --- | --- |
| `v5.5` | Самая новая версия, если она доступна в вашем аккаунте. Используйте как основной вариант для нового сценария. |
| `v5` | Новая версия Suno для качественной генерации песен и инструменталов. |
| `v4.5` | Стабильная версия для генерации песен, когда нужна совместимость с текущими настройками бота. |
| `v4` | Совместимая версия для старых сценариев Suno. |
| `v3.5` | Более старая версия для сценариев, где результат уже был настроен под неё. |
| `v3` | Старый совместимый вариант. Используйте только если сценарий завязан на эту версию. |

## Параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота. В PuzzleBot обычно используется `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота для авторизации запроса. |
| `user` | string | Да | ID пользователя или сессии. В PuzzleBot обычно используется `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Всегда `suno`. Не заменяйте его на значение submodel. |
| `submodel` | string | Да | Версия Suno: `v5.5`, `v5`, `v4.5`, `v4`, `v3.5` или `v3`. |
| `prompt` | string | Да | Описание трека, текст песни или задача для генерации. |
| `mode` | string | Нет | Метод работы Suno: `simple`, `custom`, `extend`, `separate`. |
| `styles` | string | Нет | Стили, жанры и теги через запятую. |
| `send_answer` | boolean | Нет | `true` отправляет результат пользователю. `false` сохраняет результат в `{{tracker_answer}}`. |

## Ответ

```json
{
  "answer": "Ссылка на готовый аудиофайл."
}
```
