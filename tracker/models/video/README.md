# Видео-модели

Видео-модели для генерации роликов по тексту и сценам.

## Спецификация

- Метод: `GET` — проверка доступности и статуса.
- Метод: `POST` — основной рабочий запрос.
- Базовые параметры: `prompt`, `bot`, `user`.
