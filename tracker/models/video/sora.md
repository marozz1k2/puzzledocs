# sora

Видео-модель Sora для генерации роликов по текстовому описанию или одному изображению-референсу.

## Как отправлять запрос

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Метод: `POST`
- Вид запроса в PuzzleBot: `Сформированный`
- Формат тела: `application/json` или те же ключи в no-code параметрах.

Обязательная база для любого запроса: `bot`, `token`, `user`, `model`. Дальше добавьте описание ролика в `prompt`; если нужен image-to-video, передайте одно изображение в `images`.

## Пример запроса с комментариями

```jsonc
{
  "bot": "{{BOT_USERNAME_TEXT}}", // обязательно: username бота в PuzzleBot.
  "token": "[Ваш API-токен]", // обязательно: API-токен бота для доступа к трекеру.
  "user": "{{USER_ID_TEXT}}", // обязательно: ID пользователя или сессии, к которой относится запрос.
  "model": "sora", // обязательно: точный ключ модели.
  "prompt": "{{prompt}}", // обязательно: описание сцены, движения и стиля видео.
  "images": "https://example.com/reference.jpg", // необязательно: одно изображение-референс, без массива `[]`.
  "params": { // необязательно: вложенные настройки конкретной модели.
    "aspect_ratio": "9:16", // необязательно: формат кадра `9:16` или `16:9`.
    "size": "720x1280", // необязательно: размер результата, если доступен в вашем тарифе.
    "seconds": 12 // необязательно: длительность ролика в секундах.
  }
}
```

## Параметры

| Параметр | Тип | Обязательный | Описание |
| --- | --- | --- | --- |
| `bot` | string | Да | Username бота. В PuzzleBot обычно используется `{{BOT_USERNAME_TEXT}}`. |
| `token` | string | Да | API-токен бота для авторизации запроса. |
| `user` | string | Да | ID пользователя или сессии. В PuzzleBot обычно используется `{{USER_ID_TEXT}}`. |
| `model` | string | Да | Точный ключ вызываемой модели: `sora`. |
| `prompt` | string | Да | Описание видео: сцена, действие, движение камеры, стиль и важные детали. |
| `images` | string | Нет | До 1 изображения-референса: ссылка, file_id или переменная PuzzleBot. |
| `params.aspect_ratio` | string | Нет | Формат кадра: `9:16` или `16:9`; по умолчанию для `sora` используется вертикальный формат `9:16`. |
| `params.size` | string | Нет | Размер результата, например `720x1280`; доступность зависит от текущей конфигурации модели. |
| `params.seconds` | number | Нет | Длительность ролика; в текущей конфигурации обычно используется `12`. |
| `role` | string | Нет | Дополнительная инструкция для модели: стиль, ограничения, тональность. |
| `send_answer` | boolean | Нет | `true` отправляет результат пользователю. `false` не отправляет сообщение и сохраняет результат в `{{tracker_answer}}`. По умолчанию `true`. |

## Ответ

```json
{
  "answer": "Ссылка на готовое видео."
}
```

Если `send_answer = false`, результат не отправляется пользователю в чат, а сохраняется в переменную `{{tracker_answer}}`.
