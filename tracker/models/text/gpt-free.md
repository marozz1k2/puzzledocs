# gpt_free

## Описание

`gpt_free` — текстовая модель для базовой генерации и обработки текстовых запросов в трекере PuzzleAI.

Подходит для:

- коротких ответов;
- генерации идей;
- черновиков текстов;
- простых задач суммаризации.

## Endpoint

`https://api.pxsto.re/main/puzzlebot-tracker`

## Метод

`POST`

## Параметры

| Параметр | Тип | Обязательный | Пример | Описание |
| --- | --- | --- | --- | --- |
| user | string | Да | `{{USER_ID_TEXT}}` | Идентификатор пользователя в трекере |
| model | string | Да | `gpt_free` | Системное имя модели |
| bot | string | Да | `{{BOT_ID}}` | Идентификатор бота/сценария |
| prompt | string | Да | `Напиши 5 идей для поста про ИИ` | Текстовый запрос к модели |
| temperature | number | Нет | `0.7` | Степень вариативности ответа |

{% hint style="info" %}
Если `temperature` не передан, используется значение по умолчанию на стороне сервиса.
{% endhint %}

## Пример no-code запроса

Сформированный запрос:

- Ссылка: `https://api.pxsto.re/main/puzzlebot-tracker`
- Тип запроса: `POST`
- Параметры:
  - `user`: `{{USER_ID_TEXT}}`
  - `model`: `gpt_free`
  - `bot`: `{{BOT_ID}}`
  - `prompt`: `Составь краткий контент-план на 7 дней для Telegram-канала про нейросети`
  - `temperature`: `0.7`

## Пример cURL

```bash
curl -X POST 'https://api.pxsto.re/main/puzzlebot-tracker' \
  -H 'Content-Type: application/json' \
  -d '{
    "user": "{{USER_ID_TEXT}}",
    "model": "gpt_free",
    "bot": "{{BOT_ID}}",
    "prompt": "Составь краткий контент-план на 7 дней для Telegram-канала про нейросети",
    "temperature": 0.7
  }'
```

## Ожидаемый ответ

Пример (может отличаться в зависимости от конфигурации сценария):

```json
{
  "ok": true,
  "model": "gpt_free",
  "result": "День 1: ..."
}
```

## Ошибки и диагностика

- Проверьте, что `model` строго равен `gpt_free`.
- Убедитесь, что все переменные вида `{{VAR}}` подставлены реальными значениями.
- Если ответ пустой, проверьте содержимое `prompt` и ограничения сценария бота.
